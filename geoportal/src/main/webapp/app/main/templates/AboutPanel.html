About this portal

The Interdisciplinary Earth Data Alliance (IEDA) Data Portal

This application searches a catalog of resources under the stewardship of partners in the Interdisciplinary Earth Data Alliance (IEDA). Catalog records are harvested from the EarthChem Library, US Antarctic Program Data Center, Marine Geoscience Data Library and Academic Seismic Portal partners of the Interdisciplinary Earth Data Alliance. Funding is provided by the National Science Foundation

The catalog search is implemented using the ESRI open-source Geoportal Software, v. 2.5, customized in an IEDA fork of the ESRI gitHub repository. Metadata are harvested into the catalog as ISO 19139 xml files from web accessible folders, and all metadata are available as ISO 19139 XML or JSON via links with each search result.

The catalog can be searched via API using the OGC Catalog Service for the Web (CSW), version 3.0.0 or OpenSearch. Capabilities are described in the CSW getCapabilities document and the OpenSearch description

The APIs return metadata in CSW records or Atom XML format, both of which contain basic Dublin Core metadata elements.
